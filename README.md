# Fachrin Aulia Nasution

This repository houses the source code for **[fachr.in](https://fachr.in)**, my personal web page. It reflects my philosophy of building software: **simple, performant, and scalable**.

## ðŸ›  Technical Overview

This project is built with modern web standards.

-   **Core:** [Next.js 14](https://nextjs.org/) (App Router) for server-side rendering and optimal performance.
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/) for a utility-first, maintainable design system.
-   **Content:** [MDX](https://mdxjs.com/) to treat content as data, allowing rich interactivity in blog posts.
-   **Infrastructure:** Self-hosted on **Kubernetes (K8s)**, containerized with **Docker**, and deployed via **GHCR**.
